Passivbot is an automated cryptocurrency trading bot written in Python and Rust, designed to trade perpetual futures markets with minimal user intervention. It places and manages grid and trailing limit orders, dynamically selects the most volatile coins via its Forager system, and includes a powerful backtester and multi-objective optimizer to help you find optimal configurations.

## How Passivbot works

Passivbot is a contrarian market maker. Rather than predicting price direction, it provides liquidity by placing limit orders above and below the current price. When prices move against a position, it averages down (for longs) or up (for shorts) using a configurable grid or trailing strategy.
